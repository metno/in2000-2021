# Case 6. Åpent case

Fritt valg, men må bruke minst én datakilde fra MET.

Eksempler:

- Droneflyving
- Luftkvalitet
- Klima (Frost, KSS)
- Vindvarsel for seiling og windsurfing
- Bookingberedskap for høyfjellshoteller og badesteder (se på langtidsvarsel og årlig tendens)
